自进化实验报告

基于两个自进化（self-evolve）对照实验，考察智能体在同一房间内连续 20 局的跨局学习效果。 核心问题：随着对局增多、复盘记忆不断积累，自进化阵营的胜率、白天投票质量与身份推断能力是否随之提升。

实验设置

实验 实验 ID 自进化方 对手 房间数 × 每房间局数
好人自进化 exp_38fb408448ed self_evolve_v1 好人阵营 static_strategy_v1 静态狼人 8 × 20 = 160
狼人自进化 exp_ca30ccd344dd self_evolve_v1 狼人阵营 static_strategy_v1 静态好人 8 × 20 = 160
  • 角色配置：8 人局，2 狼人、1 预言家、1 女巫、1 猎人、3 村民。
  • 分阶段口径：S1=1–5、S2=6–10、S3=11–15、S4=16–20，每段汇总 8 个房间共 40 局。
  • 自进化方在每局结束后进行复盘，将经验写入策略记忆库，供后续对局检索复用。
  • 对照组使用 static_strategy_v1（固定策略包，无局后复盘），实验变量为「固定策略包 vs 固定策略包 + 局后复盘 + room memory」。

1分阶段胜率

好人自进化效果显著：胜率从 47.5% 单调上升到 75.0%，反超 50% 均势线，且呈现清晰的随局数递增趋势。

狼人自进化几乎无增益：胜率始终在 40–45% 区间波动，S4 相比 S1 仅 +2.5pp，未形成稳定上升趋势。

两方角色配置对称，差异说明：自进化的学习红利在好人阵营上能转化为胜率，在狼人阵营上则难以转化。

机制解读：狼人胜负本质≈「两只狼是否都活到终局」。伪装变好只能减少白天被票出局，但神职夜间反制（预言家查验、女巫毒药）不受话术影响，狼人每局击杀好人数也未提升，因此暴露率的小幅改善没有传导到胜率。

3身份推断 masked accuracy 的分阶段趋势

masked accuracy 指屏蔽掉公开信息与确定性结果（如自我推断、已公开身份、预言家已查验目标）后的身份推断正确率， 更能反映智能体基于发言、投票、事件的纯推理能力。按 day=1 与 day≥2 分组观察。

高价值策略高度集中在身份起跳/对跳处理、归票纪律、神职隐藏与保护等中后期决策上，与前三节定量结论一致。

总体结论

1. 好人自进化是成功的：胜率 +27.5pp、投中狼人率 +14.2pp，且身份推断在中后期明显变强，三条证据链一致。

2. 狼人自进化收益有限：仅学到轻微伪装（暴露率 −3.7pp），但受限于狼人胜负机制，未能转化为胜率（+2.5pp）。

3. 自进化主要提升「局内中后期」的推理质量：第1天之后的 masked accuracy 跨局增益全面大于第1天，说明记忆复用在信息更丰富的中后期才能充分发挥作用。
