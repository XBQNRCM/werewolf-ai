Werewolf-AI

面向多智能体协作、博弈与自进化的狼人杀系统

我们构建了一套可观测、可消融、可跨局自进化的多智能体狼人杀系统。游戏系统提供权威规则状态机、服务端信息隔离与统一动作协议； Agent 作为独立决策服务，基于可见状态完成身份推断、策略召回、合法行动决策与终局复盘； 实验平台负责批量编排对照实验、断点续跑与全链路可观测分析。 基于这套闭环，我们验证了跨局策略记忆对好人阵营胜率、投票质量与中后期推理能力的显著提升。

1系统构成

系统拆成三个边界清晰、可独立演进的服务：游戏系统保证规则与事实可信，Agent 负责玩家认知与行动，实验平台负责批量验证效果。

game-system对局事实源

负责房间与多局生命周期、规则状态机、角色技能调度、合法行动生成、玩家视角裁剪、事件流、状态快照与回放。它提供所有对局事实的权威来源：决定当前局面、可见信息和合法动作，不参与 Agent 推理、策略记忆或实验分组。

agent认知与决策服务

每个 AI 玩家以独立会话运行，基于游戏系统下发的可见状态组织决策上下文，完成身份推断、策略召回、合法行动决策与终局复盘，并维护可追溯的跨局策略记忆和 LLM 调用记录。

experiments对照实验与分析平台

支持多房间并行、多局串行、多 Agent 配置、多模型与多角色设置的批量实验；提供断点续跑、失败恢复、实时观战、细粒度回放与多切面离线分析。它只负责编排和分析，不复制局内事实。

2核心设计

这些设计把“能跑完一局”推进到“可验证、可复盘、可持续改进”的 Agent Team 实验系统。

视角隔离

每个玩家只看到该看的信息

公开视角、玩家视角和复盘视角都由服务端统一生成。无论是前端、Agent 还是回放接口，都不能绕过后端裁剪拿到隐藏身份、夜间行动或私有技能结果。

统一协议

真人、AI 和实验脚本共用动作入口

所有参与者都先读取当前合法动作，再提交结构化行动；规则校验始终在游戏系统完成。因此同一套对局内核既能支撑人机混战，也能支撑批量自动实验。

配置实验

用配置切换 Agent 能力

是否做身份推断、是否把推断用于行动、是否使用固定策略、是否开启局后复盘，都通过配置切换。同一套运行时代码即可支撑 baseline、自进化和多模型对照。

可审计

模型当时看到了什么可以回放

发给模型的信息会按规则说明、可见事件、身份判断、策略经验等来源整理，并记录每次真实调用。出错时可以回看输入边界，而不是只看到一个最终动作。

推断决策

先判断身份，再选择行动

Agent 会先对可见玩家做身份判断和置信度估计，再在合法动作集合内选择发言、投票或技能目标。推断结果既能单独评估，也能注入决策做消融对比。

跨局进化

复盘沉淀经验，但不放任模型自改

终局复盘只能写入可追溯的策略记忆，不直接改代码、规则或固定策略包；记忆是否有用由后续对局反馈更新。下一局再按角色和阶段召回高价值经验。

3主要结果

两个自进化对照实验各 8 房间 × 20 局（共 160 局），考察同一房间内连续对局的跨局学习效果。

+27.5pp 好人自进化胜率 S4 − S1（47.5% → 75.0%）
+14.2pp 好人白天投中狼人率提升
+2.5pp 狼人自进化胜率 S4 − S1（几乎无增益）

① 好人自进化有效。胜率、投票命中率、中后期身份推断三条证据链一致上升。

② 狼人自进化收益有限。伪装略有改善，但受胜负机制约束，未能转化为胜率。

③ 增益集中在中后期。记忆复用在信息更丰富的 day≥2 阶段才充分发挥作用，而非开局首日。
